The Fibonacci sequence is defined

$$F_0 = 1, F_1 = 1$$

and

$$F_n = F_{n-1} + F_{n-1}$$

for \(n > 1\).

fib :: Int -> Int
fib 0 = 0
fib 1 = 1
fib n = fib (n-1) + fib (n-2)
